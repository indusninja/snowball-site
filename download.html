PC Requirements

The game must run on PCs which minimum requirements are identical to these of UDK:

- Windows XP SP2 or Windows Vista

- 2.0+ GHz processor

- 2 GB system RAM

- SM3-compatible video card

- 3 GB Free hard drive space

LAN access is another requirement as the game is multiplayer-only.

Install and play instructions

Once the game is installed from this, the game is ready to be played. In order to play the procedure is:

- Find host computer's IP Address

- Open the "Start Menu" > "Snowball Fight"

- Select "Host Game"

- Click on "Ready" to start game

- Add Clients on other computers by "Start Menu" > "SnowBall Fight" > "Join Game" > Enter Host Computer's IP Address > "Ready"

- Repeat procedure to add more clients.

Have fun!
